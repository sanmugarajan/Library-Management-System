# Library Management System

## Overview

Library Management System developed in C using Singly Linked Lists, Dynamic Memory Allocation and File Handling.

The application allows librarians to manage books, issue books to users, return books, maintain records and store data permanently using CSV files.

## Features

* Add New Book
* Update Book Details
* Remove Book
* Search Book
* View All Books
* Issue Book
* Return Book
* List Issued Books
* Save Records
* Load Records
* Issue Date & Time Tracking
* Return Date & Time Tracking

## Technologies Used

* C Programming
* Singly Linked List
* Dynamic Memory Allocation
* File Handling
* CSV Storage
* Date & Time Handling
* Makefile

## Compilation

```bash
make
```
